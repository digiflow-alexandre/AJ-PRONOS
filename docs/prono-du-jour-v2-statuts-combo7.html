Mes recommandations :
  • Statuts : on garde tous les 5 états (pending / live pulse / win / loss / void). Le pulse sur le live est subtil mais signal fort. Les win/loss ont un fond très subtil pour pas saturer la home.
  • Combo long : traitement B (preview 3 + collapse) est le plus adapté pour l'Accueil. Garde la card compacte (~210px) tout en laissant l'utilisateur étendre s'il veut le détail. Le A est OK pour 4-5 sélections mais devient long à 7+. Le C demande un scroll horizontal qui n'est pas évident à découvrir sur mobile.
  • Note : la card "Voir les détails" ouvre la fiche complète /bet/[id] avec toutes les sélections détaillées de toute façon — le preview 3 sur l'Accueil suffit largement à donner envie de cliquer.
