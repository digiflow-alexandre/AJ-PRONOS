Les 7 types — état du câblage

USER · or · câblée 🔥 Nouveau prono simple Envoyée à tous les abonnés dont le pack ≥ min_tier du pari, dès qu'un pari simple est publié via l'admin.
USER · violet · câblée 🎯 Nouveau combiné N sélections Variante du push "nouveau prono" mais avec le wording adapté (titre = "Combiné N sélections", body = cote totale agrégée).
USER · vert · câblée 🏆 Prono / Combiné GAGNÉ Envoyée par track-results dès qu'un pari passe à win. Inclut le score final (single) ou le ratio sélections gagnantes (combiné).
USER · rouge · câblée ❌ Prono / Combiné PERDU Même déclencheur que la gagnée, mais quand le résultat passe à loss. Body précise ce qui a planté (score, sélection ratée).
VIP · bleu · à câbler 💎 Message dans le salon VIP Switch présent dans les préférences (vip_new_messages + vip_mentions) mais aucun trigger backend ne push pour l'instant. À brancher : trigger Supabase sur insert dans chat_messages.
USER · crème · à câbler 📒 Mets à jour ton carnet Préférence daily_recap existe, mais pas de cron qui envoie. Idée : cron quotidien le soir, push aux users qui ont des paris notés sans résultat saisi.
ADMIN · orange · câblée ⏰ N paris à valider 2 sources : (1) track-results push immédiat quand le parser n'a pas compris une prédiction ; (2) cron admin-pending-reminder qui rappelle les paris en attente. Toi + Julien uniquement.
