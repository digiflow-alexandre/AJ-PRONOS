Différences vs version actuelle

  • Header noir avec logo or au lieu de juste du texte → identité immédiate
  • Eyebrow doré avant le titre (pattern magazine éditorial)
  • Title XXL avec accent doré sur 1 mot (style Substack/Linear)
  • Pack card crème avec liseré or → l'info clé saute aux yeux
  • Listes numérotées en cercles avec contraste noir/or au lieu de simples bullets
  • Cards de mise en garde colorées (vert pour OK, jaune pour warning, rouge pour danger)
  • CTA noir avec liseré or (cohérent avec la DA)
  • Footer crème avec liens + disclaimer 18+ standardisé

Notes techniques

  • Code HTML inline (compatible Gmail / Apple Mail / Outlook)
  • Pas de webfont externe (typo système pour compat max)
  • Pas d'image distante (logo dessiné en CSS — pas de blocage anti-tracking)
  • Largeur fixe 560px, responsive sur mobile
