ANALYSE GLOBALE

Comportement souhaité

  • Page combiné = juste la liste des sélections, avec sur chacune un bouton "Choisir un match à venir" tant qu'elle est vide
  • Tap sur le bouton → sous-flow plein écran en 3 étapes pour cette sélection : Match → Conditions → Cote
  • Stepper en haut dans le sous-flow : "1 · Match" / "2 · Conditions" / "3 · Cote" (l'étape active en or, ✓ pour celles déjà passées)
  • À la fin du sous-flow → retour à la page combiné avec la sélection N complétée (résumé Match + Conditions + Cote + bouton modifier)
  • Pareil pour Sélection 2, 3…
  • Header en haut : cote combinée live (produit des cotes — calcul auto) — empty si une cote manque
  • Bouton "Continuer →" en bas — désactivé tant qu'une sélection est incomplète
  • Étape 2 = même que single (récap doré + diffusion + analyse + publier)

Variantes éditables / réutilisation

  • Tap sur "✏️ Modifier" d'une sélection complétée → relance le sous-flow à l'étape "Cote" (avec back possible vers Conditions / Match)
  • Tap sur 🗑 → supprime la sélection (impossible si on n'a que 2 sélections — minimum requis)
  • "Saisir manuellement" = fallback si le match n'est pas dans l'API
